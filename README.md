## Тестовые сценарии

* Выпадающий список в разделе «Вопросы о важном». Нужно проверить: когда нажимаешь на стрелочку, открывается соответствующий текст.
* Заказ самоката. Нужно проверить весь флоу позитивного сценария с двумя наборами данных. Проверить точки входа в сценарий, их две: кнопка «Заказать» вверху страницы и внизу.
Из чего состоит позитивный сценарий:
Нажать кнопку «Заказать». На странице две кнопки заказа.
Заполнить форму заказа.
Проверить, что появилось всплывающее окно с сообщением об успешном создании заказа. Нужно написать тесты с разными данными: минимум два набора.
* Если нажать на логотип Самоката, попадёшь на главную страницу Самоката. 
* Если нажать на логотип Яндекса, в новом окне откроется главная страница Яндекса.

## Структура проекта


*   `MainPage.java` - Page Object главной страницы.
*   `OrderMakePage.java` - Page Object страницы оформления аренды самоката.
*   `FaqSteps.java` - Класс, содержащий шаги, требуемые для тестирования блока "Вопросы о важном".
*   `LogoSamokatSteps.java` - Класс, содержащий шаги, требуемые для тестирования перехода на главную страницу при клике на логотип "Самокат".
*   `LogoYandexSteps.java` - Класс, содержащий шаги, требуемые для тестирования перехода на главную страницу Яндекса при клике на логотип "Яндекс".
*   `OrderMakeSteps.java` - Класс, содержащий шаги, требуемые для тестирования оформления аренды самоката.
*   `BaseTest.java` - Класс, описывающий общие для всех тестов действия.
*   `FaqTest.java` - Тест блока "Вопросы о важном".
*   `LogoSamokatTest.java` - Тест клика по логотипу "Самокат". 
*   `LogoYandexTest.java` - Тест клика по логотипу "Яндекс".
*   `OrderMakeTest.java` - Тест оформления аренды самоката по клику верхней и нижней кнопок "Заказать".
